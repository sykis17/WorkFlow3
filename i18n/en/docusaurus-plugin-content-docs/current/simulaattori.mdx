# Rule Calculator

Tämä työkalu auttaa määrittämään, mitkä säädökset koskevat alustasi. Vastaa alla oleviin kysymyksiin saadaksesi juridisen arvion.

:::info Ohje
Tämä simulaattori on demoversio päätöksentekologiikasta. Valitse vaihtoehdot, jotka parhaiten kuvaavat alustasi tilannetta.
:::

<RuleSimulator />

---

## Miten tämä toimii?

Tämä simulaattori hyödyntää **päätöspuuta (Decision Tree)**. 
- Jokainen valinta vie käyttäjän uudelle "solmulle" (node).
- Logiikka on erotettu käyttöliittymästä, mikä tekee sääntöjen päivittämisestä helppoa.
- Voit tarkastella koodia `src/components/RuleSimulator/index.js` -tiedostosta.
