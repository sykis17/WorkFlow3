# Flight Instrument System

Tämä on dynaaminen mittaristo, joka simuloi lentokoneen perusmittareita. 

<FlightDeck />

## Käyttöohjeet:
- **W / S**: Pitch (Nokka ylös/alas)
- **A / D**: Roll (Kallistus)
- **Nuoli Ylös/Alas**: Moottorin teho
- **Nuoli Vasen/Oikea**: Yaw (Kääntyminen)
